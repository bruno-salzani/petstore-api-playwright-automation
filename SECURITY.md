## Política de Segurança

- Use versões suportadas de Java (17 ou 21) e Maven (3.9+).
- Vulnerabilidades de dependências são verificadas pelo OWASP Dependency-Check no CI.
- Reporte problemas de segurança abrindo uma issue marcada como `security`.
- Não inclua segredos em código, logs ou variáveis commitadas.
